# Colourflex Inks & Coatings Ltd.

Premium corporate website for Colourflex Inks & Coatings Ltd., Tanzania's leading manufacturer of high-performance flexographic and gravure printing inks.

## Overview

This application is a modern, responsive corporate portal designed to showcase Colourflex's manufacturing capabilities, product range, and commitment to sustainability. Built with a focus on "Premium Industrial Aesthetics," it uses a clean design language inspired by the CMYK color process.

## Key Features

- **Modern Hero Experience**: Dynamic full-screen slider with motion-enhanced content.
- **Product Catalog**: Comprehensive listing of flexographic, gravure, and screen printing solutions with technical specifications.
- **Interactive Team Profiles**: Dynamic team section with member bios and profile modals.
- **Sustainability & Ethics**: Dedicated section highlighting the company's commitment to water-based, eco-friendly inks.
- **Service & Support**: Technical support request forms and laboratory matching services.
- **Real-time Engagement**: Integrated WhatsApp widget for instant client communication.
- **Responsive Layout**: Fully optimized for mobile, tablet, and desktop viewing.

## Tech Stack

- **React 19**: Modern component-based architecture.
- **Tailwind CSS**: Utility-first styling with a custom "CMYK" inspired color palette.
- **Framer Motion**: Smooth scroll animations and element transitions.
- **Lucide React**: High-quality, consistent iconography.
- **React Router 7**: Optimized client-side navigation.

## Design Philosophy

The website follows a "Rich Industrial" theme:
- **Colors**: Deep blacks (`#111111`) paired with vivid Process Cyan (`#009FE3`) and Magenta (`#EC008C`) accents.
- **Typography**: Plus Jakarta Sans for a modern, accessible, yet professional feel.
- **Interactions**: Subtle parallax effects, backdrop-blur components, and soft premium shadows.
